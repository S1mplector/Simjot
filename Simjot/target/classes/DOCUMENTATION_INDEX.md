## Compliance

All assets and libraries used in Simjot comply with their respective licensing terms:
- Proper attribution where required
- No commercial use violations
- Preservation of copyright notices
- Appropriate license compatibility

## For Developers

When adding new third-party assets or libraries:
1. Create a corresponding notice file in this directory
2. Update this index file
3. Ensure license compatibility with the main project
4. Include proper attribution and usage rights

## Contact

For questions about licensing, usage rights, or compliance:

- Contact project maintainers for clarification
