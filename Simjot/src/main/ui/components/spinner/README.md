# Spinner Components

Modern numeric input components with stepping functionality.

## Components

- **ModernSpinner**: Numeric input with stepping and validation
- **ModernSpinnerUI**: Custom UI delegate for modern appearance

## Features

- Numeric input validation
- Smooth stepping animations
- Theme-aware design
- Keyboard navigation
- Custom number formats

## Usage

```java
ModernSpinner spinner = new ModernSpinner();
spinner.setMinimum(0);
spinner.setMaximum(100);
spinner.setStepSize(1);
spinner.setValue(50);
```
