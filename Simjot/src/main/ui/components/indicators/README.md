# Indicator Components

Visual feedback and status indicator components.

## Components

- **SaveIndicatorPanel**: Auto-save status display with animations
- **QuickMoodWidget**: Mood visualization and input component

## Features

- Real-time status updates
- Smooth animations
- Theme-aware design
- Accessibility support
- Compact display

## Usage

```java
SaveIndicatorPanel indicator = new SaveIndicatorPanel();
indicator.setStatus(SaveStatus.SAVING);
indicator.setAnimationEnabled(true);
```
