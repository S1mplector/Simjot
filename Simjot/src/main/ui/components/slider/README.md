# Slider Components

Modern slider components with smooth animations and custom styling.

## Components

- **MoodSlider**: Emotional state input with visual feedback

## Features

- Smooth sliding animations
- Visual mood indicators
- Theme-aware design
- Keyboard navigation
- Accessibility support

## Usage

```java
MoodSlider moodSlider = new MoodSlider();
moodSlider.setRange(1, 10);
moodSlider.setValue(5);
moodSlider.setShowLabels(true);
moodSlider.setAnimationEnabled(true);
```
