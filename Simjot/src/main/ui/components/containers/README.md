# Container Components

Modern container panels with glass-morphism effects and advanced styling.

## Components

- **AeroPanel**: Glass-morphism panels with blur effects
- **RoundedPanel**: Modern rounded corner containers
- **TranslucentPanel**: Semi-transparent overlays
- **FrostedGlassPanel**: Advanced glass effect with customizable blur

## Features

- Glass-morphism design
- Blur and transparency effects
- Theme-aware styling
- Smooth animations
- High-DPI support

## Usage

```java
AeroPanel panel = new AeroPanel();
panel.setBlurRadius(10);
panel.setOpacity(0.8f);
panel.add(content);
```
