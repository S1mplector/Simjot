# Field Components

Modern input field components with validation and styling.

## Components

- **TitleDividerField**: Title field with integrated divider
- **ModernTextField**: Sleek text input with validation

## Features

- Input validation
- Placeholder text
- Theme-aware styling
- Error states
- Accessibility support

## Usage

```java
TitleDividerField titleField = new TitleDividerField();
titleField.setPlaceholder("Enter title...");
titleField.setRequired(true);
```
