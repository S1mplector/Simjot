## Simple English Dictionary

This application incorporates data from the Simple English Dictionary project, which is used for enhanced synonym and rhyme detection capabilities in our poetry analysis tools.

**Repository:** https://github.com/adrahtz/simple-english-dictionary  
**License:** The Simple English Dictionary project is licensed under the MIT License  
**Author:** adrahtz  
**Description:** A lightweight JSON-based English dictionary providing word definitions, synonyms, and related linguistic data that powers the intelligent poetry assistance features in Simjot.

We extend our gratitude to the Simple English Dictionary project for providing this valuable linguistic resource that enhances the poetry writing and analysis experience.
