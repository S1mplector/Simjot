# ComboBox Components

Modern dropdown selection components with enhanced styling and functionality.

## Components

- **ModernComboBox**: Dropdown selection with custom rendering
- **ModernComboBoxUI**: Custom UI delegate for modern appearance

## Features

- Custom popup styling
- Smooth animations
- Theme-aware design
- Keyboard navigation
- Custom item rendering

## Usage

```java
ModernComboBox<String> comboBox = new ModernComboBox<>();
comboBox.setItems("Option 1", "Option 2", "Option 3");
comboBox.setPlaceholder("Select an option");
```
