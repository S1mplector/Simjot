# Toolbar Components

Specialized toolbar implementations for different UI contexts.

## Components

- **PoetryStyleToolbar**: Poetry formatting tools with rich text support

## Features

- Rich text formatting (Bold, Italic, Underline, Strikethrough)
- Theme-aware styling
- Customizable controls
- Lightweight rendering
- Editor integration

## Usage

```java
PoetryStyleToolbar toolbar = new PoetryStyleToolbar();
toolbar.setTitle("Poetry Editor");
toolbar.addCustomControl(exportButton);
toolbar.setThemeVariant(Theme.Variant.AERO);
```
