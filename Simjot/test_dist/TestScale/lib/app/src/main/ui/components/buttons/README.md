# Button Components

Modern button implementations with enhanced styling and animations.

## Components

- **MainMenuButton**: Primary navigation buttons with hover effects
- **RoundedButton**: Modern action buttons with rounded corners
- **RoundedToggleButton**: State toggle buttons with smooth transitions
- **ToolbarIconButton**: Icon-based toolbar actions
- **IconMenuButton**: Icon menu items with dropdown support

## Features

- Smooth hover animations
- Theme-aware styling
- Accessibility support
- Customizable shapes and colors
- GPU-accelerated rendering

## Usage

```java
RoundedButton button = new RoundedButton("Click Me");
button.setStyle(ButtonStyle.PRIMARY);
button.setAnimationEnabled(true);
```
