# Input Components

Advanced input components with validation and modern styling.

## Components

- **AeroTextField**: Aero-styled text fields with glass effects
- **AeroPasswordField**: Secure password entry with modern design

## Features

- Glass-morphism design
- Input validation
- Placeholder text
- Theme-aware styling
- Security features

## Usage

```java
AeroTextField textField = new AeroTextField();
textField.setPlaceholder("Enter text...");
textField.setValidationMode(ValidationMode.ON_INPUT);
textField.setThemeVariant(Theme.Variant.AERO);
```
