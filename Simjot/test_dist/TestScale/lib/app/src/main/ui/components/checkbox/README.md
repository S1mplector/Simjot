# Checkbox Components

Custom checkbox implementations with modern styling and animations.

## Components

- **ModernCheckBox**: Checkbox with custom styling and smooth transitions
- **AeroCheckBoxUI**: Aero-themed checkbox appearance delegate

## Features

- Custom visual design
- Smooth state transitions
- Theme-aware appearance
- Accessibility support
- Keyboard navigation

## Usage

```java
ModernCheckBox checkBox = new ModernCheckBox("Enable feature");
checkBox.setThemeVariant(Theme.Variant.AERO);
checkBox.setAnimationEnabled(true);
```
