# Popup Components

Animated dialog and popup components with glass effects.

## Components

- **AnimatedGlassPopup**: Smooth animated dialogs with glass-morphism

## Features

- Smooth fade animations
- Glass-morphism design
- Modal blocking
- Keyboard navigation
- Theme integration

## Usage

```java
AnimatedGlassPopup popup = new AnimatedGlassPopup();
popup.setAnimationDuration(400);
popup.setEasingFunction(EasingFunction.EASE_OUT_CUBIC);
popup.show(content);
```
