# Editor Components

Rich text editor components with advanced formatting capabilities.

## Components

- **RichTextStyler**: Advanced text formatting and styling
- **EditorUIUtils**: Editor helper functions and utilities

## Features

- Rich text formatting
- Style management
- Text processing utilities
- Editor integration helpers

## Usage

```java
RichTextStyler styler = new RichTextStyler();
styler.applyStyle(textPane, Style.BOLD);
styler.setAttributes(font, color);
```
