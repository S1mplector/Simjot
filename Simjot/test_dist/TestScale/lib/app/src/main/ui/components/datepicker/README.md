# Date Picker Components

Modern date selection component with calendar interface.

## Components

- **ModernDatePicker**: Calendar date selection with modern styling

## Features

- Calendar interface
- Theme-aware design
- Keyboard navigation
- Date validation
- Internationalization support

## Usage

```java
ModernDatePicker datePicker = new ModernDatePicker();
datePicker.setDate(LocalDate.now());
datePicker.setDateFormat(DateFormat.MEDIUM);
```
